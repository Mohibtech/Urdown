# خوش آمدید!!!
اُرڈاوُن (`Urdown`) انٹرنیٹ پر مبنی ٹیکسٹ ایڈیٹر ہے۔ اِس کے ذریعے آپ اُردو متن کو
`HTML` میں تبدیل کر سکتے ہیں۔ اردو میں ٹایٔپ کرنے  کے لیٔے [گُوگل کروم ایکسٹینشن](https://www.google.com/inputtools/try/) یا [اردو فونیٹِک کِیبورڈ](http://urdu.ca/1) اپنے کمپیوٹر میں ڈالیں۔

مثال کے طور پر آپ:
* **متن کا وزن بڑھا سکتے ہیں**۔
* *متن کو تِرچھا کر سکتے ہیں*۔
* متن کو [دوسری وِیب سایٔٹ سے جوڑ سکتے ہیں۔](http://iahmed.me)

اِس کے علاوہ آپ انگریزی میں بھی متن ڈال سکتے ہیں:

،،،
Urdown is a web-based markdown editor for Urdu. The source is available on
[Github](https://github.com/hazrmard/Urdown).  
  
Install [Google Input Tools Extension](https://www.google.com/inputtools/try/) on  Chrome or [Urdu Phonetic Keyboard](http://urdu.ca/1) on your computer to begin typing in Urdu.
  
This is some english text. English text, when enclosed in 3 commas on either side
becomes left-aligned. It follows the same markdown rules as the Urdu text. For
example, you can:
* Make text **bold like this**.
* Or make it *slanted like this*.
* [Or add a hyperlink like this.](http://iahmed.me)
